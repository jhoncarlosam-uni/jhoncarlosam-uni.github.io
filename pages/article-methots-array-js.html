Métodos Pop, Push, Shift y Unshift en JavaScript

Fecha: 10 de Enero, 2022
JavaScript nos da 4 métodos para añadir o remover elementos del comienzo o final de un arreglo:

pop(): Elimina un elemento al final de un arreglo.


let cats = ['Pandora', 'Cheo', 'Salem'];
cats.pop(); // ['Pandora', 'Cheo']

pop() devuelve el elemento eliminado.


push(): Añade elementos desde el final de un arreglo.


let cats = ['Pandora']
cats.push('Cheo'); // ['Pandora', 'Cheo']
cats.push('Salem', 'Luna'); // ['Pandora', 'Cheo', 'Salem', 'Luna']

push() devuelve la nueva longitud del arreglo.


shift(): Elimina un elemento del principio de un arreglo.


let cats = ['Pandora', 'Cheo', 'Salem']
cats.shift(); // ['Cheo', 'Salem']

shift() devuelve el elemento eliminado.


unshift(): Añade elementos al principio de un arreglo.


let cats = ['Pandora']
cats.unshift('Cheo'); // ['Cheo', 'Pandora']
cats.unshift('Salem', 'Luna') // ['Salem', 'Luna', 'Cheo', 'Pandora']

unshift() devuelve la nueva longitud del arreglo.
